# Bus Management System

## Project Description
This is a simple Java Bus Management System project that manages buses and operations using Object-Oriented Programming principles.

## Features
- Manage buses
- Object-oriented design
- Simple console-based system

## Technologies Used
- Java (JDK 17)
- Git & GitHub
- Docker (Dockerfile included)

## How to Run
```bash
java -jar BusManagementSystem.jar
